# Cursor Expansion Prompt for TEF Prep Units

## Objective
The aim is to expand the TEF (Test d'Évaluation de Français) preparation units to include B1/B2 content, along with audio assets, and to establish a robust CMS (Content Management System) and admin panel for both web and mobile applications.

## Scope
- **B1/B2 Content Expansion**: 
  - Identify key language competencies required for B1/B2 levels.
  - Design new exercises and learning modules that cater to these levels.
  - Incorporate interactive elements that facilitate learning and retention.

- **Audio Assets**:
  - Curate audio materials covering diverse topics suitable for B1/B2 learners.
  - Ensure audio quality is high and content is engaging.
  - Implement strategies for integrating audio assets into the existing curriculum seamlessly.

- **CMS/Admin Panel Development**:
  - Choose an appropriate tech stack for building the CMS/admin panel (e.g., React for frontend, Node.js for backend).
  - Design a user-friendly interface for easy content management.
  - Ensure the CMS supports both web and mobile applications, with responsive design principles.

## Implementation Steps
1. **Research and Analysis**:
   - Gather insights from educators and experienced TEF instructors about essential B1/B2 competencies.
   - Evaluate current content to identify gaps and areas for improvement.

2. **Content Development**:
   - Create new B1/B2 learning units.
   - Develop accompanying audio materials, ensuring they are culturally relevant and cover various accents.

3. **CMS Development**:
   - Set up the database schema to support content types (text, audio, quizzes).
   - Develop the frontend and backend, ensuring secure user authentication for admin features.

4. **Testing and Feedback**:
   - Conduct usability testing with real users (teachers and students) to gather feedback on both the expanded units and the CMS interface.
   - Incorporate feedback and refine the products before the final rollout.

## Considerations
- Ensure alignment with relevant educational standards and frameworks for language learning.
- Focus on user engagement and retention strategies to enhance learning outcomes.

## Conclusion
This expansion will provide learners with comprehensive resources tailored to their needs at the B1/B2 levels, while also supporting educators with an intuitive admin panel for efficient content management.
