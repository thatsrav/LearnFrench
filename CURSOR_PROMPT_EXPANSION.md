# CURSOR PROMPT EXPANSION

## Curriculum Expansion

**Prompt:**  "As a curriculum developer, I want to generate a comprehensive course outline for a new language learning module that incorporates various teaching methods. The module should focus on immersive activities, cultural insights, and practical language applications. Please provide an outline that includes objectives, key topics, and suggested materials for each section."

## Audio Management

**Prompt:**  "As an audio manager, I need to develop a set of audio resources that align with our language curriculum. Generate a list of audio files required for each module, including native speaker dialogues, pronunciation guides, and background cultural sounds. Please specify file formats and duration for each audio clip."

## CMS Development

**Prompt:**  "As a CMS developer, I want to design a structure for our language learning platform that supports multimedia content, user engagement, and assessment features. Outline the core components of the CMS, including user roles, content types, and integration with third-party tools. Provide a detailed description of the database schema required to support these features.

Feel free to modify and expand upon these prompts as needed to tailor them to your specific curriculum development needs!
